# 差异化数据呈现方案设计文档

## 一、方案概述

### 1.1 设计目标
针对招聘流程管理单表包含50+字段的特点，设计差异化数据呈现方案：
- **页面视图**：展示核心关键字段（8-12个），界面简洁、信息突出
- **数据导出**：提供完整字段（50+个），满足数据分析需求

### 1.2 核心原则
- 信息分层：核心信息优先展示，次要信息按需查看
- 用户体验：减少认知负担，提升操作效率
- 数据完整：导出时保证数据不丢失

### 2.1 技术可行性评估

| 评估维度 | 可行性 | 说明 |
|---------|-------|------|
| **前端渲染** | ✅ 完全可行 | JavaScript动态控制列显示，技术成熟 |

| **导出功能** | ✅ 完全可行 | SheetJS支持自定义字段导出 |
| **性能影响** | ✅ 可接受 | 千级数据量下性能差异可忽略 |
| **浏览器兼容** | ✅ 良好 | 现代浏览器均支持 |


## 三、用户体验影响分析

### 3.1 正面影响

| 方面 | 改善效果 |
|-----|---------|
| **信息聚焦** | 用户一眼看到关键信息，减少视觉干扰 |
| **操作效率** | 核心操作（处理/查看）更直观，减少滚动 |
| **认知负担** | 降低信息过载，新手更容易上手 |
| **响应速度** | 页面渲染更快，交互更流畅 |

### 3.2 潜在问题与解决方案

| 潜在问题 | 影响程度 | 解决方案 |
|---------|---------|---------|
| 用户需要查看非核心字段 | 中 | 提供"查看详情"弹窗展示完整信息 |
| 不同用户关注字段不同 | 中 | 提供字段自定义显示功能 |
| 导出数据与页面显示不一致 | 低 | 明确标识导出为"完整数据" |

### 5.2 字段筛选逻辑实现

```javascript
/**
 * 字段筛选器
 * 根据视图类型返回对应的字段配置
 */
class FieldFilter {
  constructor(config) {
    this.config = config;
  }
  
  /**
   * 获取列表展示字段
   */
  getListFields() {
    return this.config.core;
  }
  
  /**
   * 获取详情展示字段（按分组）
   */
  getDetailFields() {
    const fields = this.config.detail;
    const groups = {};
    
    fields.forEach(field => {
      const group = field.group || 'other';
      if (!groups[group]) {
        groups[group] = [];
      }
      groups[group].push(field);
    });
    
    return groups;
  }
  
  /**
   * 获取导出字段
   */
  getExportFields() {
    return this.config.export;
  }
  
  /**
   * 根据字段名获取配置
   */
  getFieldConfig(fieldName) {
    return this.config.export.find(f => f.field === fieldName);
  }
  
  /**
   * 数据映射：将原始数据映射为展示数据
   */
  mapData(rawData, fields) {
    return rawData.map(item => {
      const mapped = {};
      fields.forEach(field => {
        mapped[field.field] = item[field.field];
        mapped[`${field.field}_label`] = field.label;
      });
      return mapped;
    });
  }
}
```

## 六、导出功能实现方案

### 6.1 导出功能设计

```javascript
/**
 * 数据导出管理器
 * 支持Excel/CSV导出，可自定义字段
 */
class ExportManager {
  constructor(fieldConfig) {
    this.fieldConfig = fieldConfig;
  }
  
  /**
   * 导出为Excel
   * @param {Array} data - 原始数据
   * @param {Object} options - 导出选项
   */
  exportToExcel(data, options = {}) {
    const {
      fields = 'all',      // 'all' | 'core' | Array<string>
      fileName = '招聘流程数据',
      sheetName = '数据'
    } = options;
    
    // 确定导出字段
    let exportFields;
    if (fields === 'all') {
      exportFields = this.fieldConfig.export;
    } else if (fields === 'core') {
      exportFields = this.fieldConfig.core;
    } else if (Array.isArray(fields)) {
      exportFields = this.fieldConfig.export.filter(f => fields.includes(f.field));
    }
    
    // 转换数据格式
    const exportData = this.transformData(data, exportFields);
    
    // 创建工作簿
    const ws = XLSX.utils.json_to_sheet(exportData);
    const wb = XLSX.utils.book_new();
    XLSX.utils.book_append_sheet(wb, ws, sheetName);
    
    // 设置列宽
    ws['!cols'] = exportFields.map(f => ({ wch: f.width || 15 }));
    
    // 下载文件
    XLSX.writeFile(wb, `${fileName}_${new Date().toISOString().split('T')[0]}.xlsx`);
  }
  
  /**
   * 转换数据格式
   * 将原始数据转换为导出格式（使用中文表头）
   */
  transformData(data, fields) {
    return data.map((item, index) => {
      const row = { '序号': index + 1 };
      
      fields.forEach(field => {
        let value = item[field.field];
        
        // 特殊字段处理
        switch (field.field) {
          case 'current_stage':
            value = this.translateStage(value);
            break;
          case 'current_status':
          case 'first_interview_result':
          case 'second_interview_result':
            value = this.translateStatus(value);
            break;
          case 'is_reported':
            value = value === 'yes' ? '是' : value === 'no' ? '否' : '-';
            break;
          case 'created_at':
          case 'updated_at':
            value = value ? new Date(value).toLocaleString('zh-CN') : '-';
            break;
          default:
            value = value || '-';
        }
        
        row[field.label] = value;
      });
      
      return row;
    });
  }
  
  /**
   * 翻译环节
   */
  translateStage(stage) {
    const map = {
      'first_interview': '初试',
      'second_interview': '复试',
      'hired': '录用',
      'onboarded': '报到'
    };
    return map[stage] || stage;
  }
  
  /**
   * 翻译状态
   */
  translateStatus(status) {
    const map = {
      'pending': '待处理',
      'pass': '通过',
      'reject': '不通过'
    };
    return map[status] || status;
  }
}
```
